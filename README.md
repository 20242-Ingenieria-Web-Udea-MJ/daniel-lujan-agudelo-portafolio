## Quickstart

1. Clone the repository

```bash
git clone https://github.com/daniel-lujan/portfolio-next.git
cd portfolio-next
```

2. Install dependencies

```bash
# npm install
# yarn install
bun install
```

3. Run in development mode

```bash
# npm run dev
# yarn run dev
bun run dev
```
